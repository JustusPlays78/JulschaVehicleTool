# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New Project |
| `Ctrl+O` | Open Project |
| `Ctrl+S` | Save Project |
| `Ctrl+Shift+S` | Save Project As |
| `Alt+F4` | Exit |

## Tree / Sidebar

| Shortcut | Action |
|----------|--------|
| `Delete` | Delete selected vehicle |
| `F2` | Rename selected item (resource or vehicle) |

## Mouse — Tree

| Action | Result |
|--------|--------|
| Click vehicle | Switch to vehicle editors |
| Click resource | Open Resource Settings |
| Click project node | Open Siren Pool editor |
| Right-click vehicle | Context menu (rename, clone, import, export, delete) |
| Right-click resource | Context menu (rename, add vehicle, import, delete) |
| Checkbox next to vehicle | Select for batch preset application |

## Mouse — 3D Viewer

| Input | Action |
|-------|--------|
| Left drag | Orbit camera |
| Right drag | Pan |
| Scroll | Zoom |
| Middle drag | Pan (alternative) |

## Mouse — Sequencer Grid (Siren Editor)

| Input | Action |
|-------|--------|
| Click a bit | Toggle on/off |
| Click + drag | Paint multiple bits at once |
